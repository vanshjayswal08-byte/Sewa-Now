## 📖 Overview
**Sewa Now** is a comprehensive full-stack service marketplace designed to seamlessly connect local service professionals (such as plumbers, electricians, and other skilled workers) with customers in need of immediate assistance. 

The platform features a dedicated Customer portal for booking services, a Worker portal for managing job requests, and a highly interactive, data-driven Admin Dashboard for overall platform management.

## ✨ Key Features

### 🧑‍💻 Admin Dashboard
* **Interactive Analytics:** Real-time data visualization using Chart.js (Line charts for booking trends, Doughnut charts for status distribution).
* **Dynamic Filtering:** Custom date-range filters to generate specific insights without page reloads.
* **Data Export:** Export charts as PNGs or download raw metric data as CSV files.
* **Record Management:** Perform seamless CRUD operations on user and worker databases.

### 👥 User & Worker Portals
* **Role-Based Access:** Secure login and session management powered by Google Firebase.
* **Service Booking:** Streamlined flow for customers to browse categories, select local workers, and track booking status (Pending, Confirmed, Completed, Cancelled).
* **Worker Profiles:** Categorized listings for local professionals to manage incoming requests.

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Data Visualization:** Chart.js (with DataLabels and Zoom plugins)
* **Backend:** Python (REST APIs)
* **Authentication & Database:** Google Firebase
